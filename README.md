# 🚀 SaipFuncs | Fizik & Matematik Simülasyonu

**SaipFuncs**, matematiksel fonksiyonları fiziksel bir ortamda simüle eden, interaktif bir öğrenme ve test platformudur. Kullanıcıların doğrusal ve karmaşık fonksiyonlar (parabol vb.) üzerinde bir nesnenin hareketini gözlemlemesine olanak tanır.

## 🛠 Öne Çıkan Özellikler

* **Dinamik Fonksiyon Motoru:**
    * **Sürükleme Modu:** $y = ax + b$ formülündeki parametreleri slider (kaydırıcı) ile anlık değiştirme.
    * **Yazma Modu:** $x^2$, $2x^2 - 3$ veya daha karmaşık fonksiyonları doğrudan metin olarak girme desteği.
* **Gelişmiş Navigasyon (Map System):**
    * Mouse tekerleği ile **Zoom (Yakınlaşma/Uzaklaşma)**.
    * Basılı tutarak serbestçe **Pan (Kaydırma)** yapabilme.
    * Tek tıkla **⌖ Merkeze Dön** özelliği.
* **Gerçekçi Fizik Motoru:**
    * Yerçekimi, sürtünme ve eğim ivmelenmesi hesaplamaları.
    * Eğimli yüzeylerde (paraboller dahil) gerçekçi kayma ve salınım hareketi.
* **Oyunlaştırma:**
    * Özel tasarlanmış **Level Sistemi** ve serbest çalışma için **Sandbox Modu**.
    * Otomatik tekrar (Auto-Restart) seçeneği.
* **Modern UI/UX:**
    * Genişletilebilir/Daraltılabilir sol kontrol paneli.
    * Sayısal verileri elle girerken odak kaybında otomatik sıfırlama (Self-Healing Input).

## 🚀 Başlangıç

Projeyi yerel makinenizde çalıştırmak için:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone [https://github.com/M3hmeT1888/SAIPFUNCS-website.git](https://github.com/M3hmeT1888/SAIPFUNCS-website.git)
    ```
2.  **Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    ```
3.  **Geliştirme Sunucusunu Başlatın:**
    ```bash
    npm run dev
    ```
